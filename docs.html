Documentation:

Version: WiseLang 1.0.5 Chutiya

Hello, World!

eww "Hello, World!";

The Terminator

All statements end with ";" i.e. the Terminator

Declaring a Variable:

WTF a = 10;
WTF b = "variable";
Note that the variable is case-sensitive. e.g. "A" is not the name as "a"

If/Else:

WTF a = 10;
agar a == 10 {
eww "a is 10";
}
nahi toh {
eww "a is not 10";
}

Pass, Break, Comments:

// Comments start with //
chutiya; // pass
hatt; // break

Operators:

Comparison Operators:

Operator Name Example
== Is Equal To x == y;
!= Is not Equal To x != y;
> Is Greater than x > y;
>= Is Greater than or equal to x >= y;
< Is Lesser than x < y;
<= Is Lesser than or equal to x <= y;

Arithmetic Operators:

Operator Name Example
+ Addition x + y;
- Subtraction x - y;
* Multiplication x * y;
/ Division x / y;
% Modulus x % y;
